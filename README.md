# E-Commerce Web Application

## Technologies

- HTML
- CSS
- JavaScript
- Python (Flask)
- SQLite

## Features

- User Registration
- User Login
- Admin Login
- Product Management
- Add Products
- Edit Products
- Delete Products
- Product Catalog
- Shopping Cart
- Checkout
- Order Tracking

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

Open

```
http://127.0.0.1:5000
```

## Folder Structure

```
templates/
static/
app.py

README.md
```
